## Coin Combinations

This project aims to find and print all the possible coin combinations of a given amount.

### Example

For the given coins find and print all the possible combinations.
EG:  
int[] coins = {1, 2, 5};  
int amount = 10;  

#### Output

5: 2, 2: 0, 1: 0  
5: 1, 2: 1, 1: 3  
5: 1, 2: 2, 1: 1  
5: 0, 2: 5, 1: 0  
5: 0, 2: 4, 1: 2  
5: 0, 2: 3, 1: 4  
5: 0, 2: 2, 1: 6  
5: 0, 2: 1, 1: 8  
5: 0, 2: 0, 1: 10  
5: 1, 2: 0, 1: 5
